Black Lives Still Matter

Project Info:

This project utilizes perceptual art, creating an immersive experience for a cause that is far more than 2-dimensional. This is a 3D sculpture. Each sphere represents a black life lost due to police violence. Viewing the 99 spheres from six meters away, you see an image of a raised fist that represents unity and solidarity in the ongoing fight against racism..

Project Details:
